# Justice Path

A Vite + React + TypeScript app with shadcn-ui and Tailwind CSS.

## Getting Started

Requirements:

- Node.js LTS
- npm

Install and run:

```sh
npm install
npm run dev
```

## Environment Variables

Create a `.env.local` in the project root with:

```ini
VITE_SUPABASE_URL=
VITE_SUPABASE_PUBLISHABLE_KEY=
```

## Tech Stack

- Vite
- React
- TypeScript
- shadcn-ui
- Tailwind CSS
